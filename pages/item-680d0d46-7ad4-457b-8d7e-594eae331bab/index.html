Tentu, mari kita tambahkan fitur menghapus data siswa ke dalam aplikasi kita.

Konsep Menghapus Data

  • Konfirmasi Penghapusan: Sebelum menghapus data, sebaiknya ditampilkan konfirmasi kepada pengguna untuk memastikan bahwa mereka ingin menghapus data tersebut.
  • Query DELETE: Kita akan menggunakan query DELETE untuk menghapus data dari database berdasarkan ID siswa.

Contoh Kode

form_edit.php (tambahkan tombol hapus)

<a href="hapus_siswa.php?id=<?php echo $row['id']; ?>" onclick="return confirm('Apakah Anda yakin ingin menghapus data ini?')">Hapus</a>
