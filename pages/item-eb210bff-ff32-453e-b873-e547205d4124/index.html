Diluncurkan pada tahun 2012, CodePen menata dirinya sebagai Front-End Developer Playground, yang bukan hanya pustaka potongan kode yang berguna, tetapi lebih sebagai tempat pengujian interaktif dan alat pembelajaran kolaboratif ( Dunn, 2018 ). Ini adalah platform pengembangan tempat siapa pun dapat mengedit bahasa front-end seperti HTML, CSS, dan JS dari browser mereka tanpa mengunduh perangkat lunak apa pun dan sebagai bagian terbaiknya, hasilnya dapat dilihat secara real-time untuk debugging yang lebih cepat ( Silva, 2021 ). Platform ini berbeda dengan pen dan proyek, di mana yang pertama hanya mendukung file HTML, CSS, dan JS tunggal dengan kurang dari 5000 baris kode, sedangkan yang terakhir mendukung beberapa file dengan lebih banyak baris (Wilkins, 2021). Langkah-langkah untuk membuat akun di CodePen diberikan.
