Memahami Prosesnya

  1. Koneksi ke Database: Seperti yang telah kita bahas sebelumnya, langkah pertama adalah membuat koneksi ke database yang ingin kita akses.
  2. Membuat Query: Kita akan membuat query SQL (Structured Query Language) untuk mengambil data yang kita inginkan. Query ini akan dikirimkan ke database.
  3. Mengeksekusi Query: Query yang telah dibuat akan dieksekusi menggunakan fungsi yang disediakan oleh ekstensi database yang kita gunakan (misalnya, mysqli_query() atau PDO::query()).
  4. Mengambil Hasil Query: Hasil dari eksekusi query akan berupa objek yang berisi data yang diambil. Kita perlu mengambil data ini dan menampilkannya dalam format yang kita inginkan.

Contoh Kode Menggunakan mysqli

&lt;?php // Koneksi ke database (asumsi koneksi sudah dibuat) // Query untuk mengambil semua data dari tabel 'produk' $sql = "SELECT * FROM produk"; $result = mysqli_query($conn, $sql); // Cek apakah query berhasil dijalankan if (mysqli_num_rows($result) &gt; 0) { // Output data dalam bentuk tabel HTML echo "&lt;table border='1'&gt; &lt;tr&gt; &lt;th&gt;ID&lt;/th&gt; &lt;th&gt;Nama Produk&lt;/th&gt; &lt;th&gt;Harga&lt;/th&gt; &lt;/tr&gt;"; // Fetch data satu per satu while($row = mysqli_fetch_assoc($result)) { echo "&lt;tr&gt;"; echo "&lt;td&gt;" . $row["id"] . "&lt;/td&gt;"; echo "&lt;td&gt;" . $row["nama_produk"] . "&lt;/td&gt;"; echo "&lt;td&gt;" . $row["harga"] . "&lt;/td&gt;"; echo "&lt;/tr&gt;"; } echo "&lt;/table&gt;"; } else { echo "Tidak ada data"; } ?&gt;
