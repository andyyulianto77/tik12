Anda memerlukan perangkat lunak di PC untuk kursus ini. Jangan khawatir, kursus ini gratis. Selain hosting, semua yang digunakan dalam kursus ini gratis. Termasuk buku teks ini.

Gratis itu bagus. Bukankah sudah saya sebutkan sebelumnya?

Serius deh, ada banyak sekali perangkat lunak gratis yang bagus di luar sana. Posting di forum Moodle jika Anda menemukan sesuatu yang benar-benar Anda sukai.

Anda dapat mencarinya di Google. Anda dapat bertanya kepada siswa lain di forum umum Moodle. Jika Anda menemukan konten Mac yang cocok untuk ditautkan ke situs ini, beri tahu saya.

Lingkungan duplikat

Anda akan menulis banyak kode. Buatlah kode yang mudah digunakan. Jalankan kode tersebut di komputer Anda terlebih dahulu, lalu transfer ke server Anda setelah kode tersebut siap.

Agar ini berfungsi paling baik, akan lebih mudah jika Anda memiliki perangkat lunak yang sama (kurang lebih) yang berjalan di PC Anda, dan server Reclaim Anda.

Itulah inti pelajaran ini.

Bagian aplikasi web

Tiga perangkat lunak utama akan bekerja sama untuk melayani aplikasi Anda:

  • Apache, sebuah server web
  • MySQL, sebuah server basis data
  • PHP, sebuah bahasa pemrograman

Anda akan mempelajari apa yang mereka lakukan nanti. Untuk saat ini, ketahuilah bahwa ini adalah hal-hal yang Anda butuhkan.

Mereka sudah disiapkan di server Reclaim Hosting Anda (dan hampir semua host murah lainnya). Yang lebih hebat lagi, Reclaim memberi Anda alat untuk membantu Anda bekerja dengan Triumphant Trio (Apache, MySQL, dan PHP), yang disebut cPanel. Anda akan cukup sering menggunakan cPanel.

Oke. Triumphant Trio berjalan di server Anda. Jadi, mari kita pasang ketiganya di PC Anda.

Beberapa saran:

  • Pada Windows, jangan instal XAMPP di C:\Program files. Instal saja di C:\xampp. Atau D:\xampp, atau apa pun. Saya taruh milik saya di D:\xampp.
  • Instal setidaknya empat hal: Apache, MySQL, PHP, dan phpMyAdmin. Instal juga fitur email palsu, jika Anda ingin mencobanya. Fitur ini akan memungkinkan Anda menulis program pengiriman email, tanpa benar-benar mengirim email.

(Oke, bagi saya, memiliki program sendiri yang mengirimkan email untuk peringatan dan semacamnya, itu cukup keren.)

XAMPP memiliki panel kontrol (CP) yang memungkinkan Anda menjalankan Apache, MySQL, dan melakukan hal-hal lainnya. Panel kontrol ini berfungsi paling baik jika Anda menjalankannya sebagai admin, terutama jika Anda ingin mengubah konfigurasi XAMPP.
