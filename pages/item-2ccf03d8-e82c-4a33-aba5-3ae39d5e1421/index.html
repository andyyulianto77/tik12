Web bekerja berdasarkan model klien/server. Yaitu saat satu komputer melakukan sesuatu untuk komputer lain.

Istilah klien dan server bisa membingungkan. Terkadang, orang berbicara tentang mesin fisik, seperti PC (klien) dan server Reclaim Anda.

Terkadang, orang-orang berbicara tentang perangkat lunak. Peramban web Anda, seperti Firefox, adalah klien, dan berkomunikasi dengan perangkat lunak server web, seperti Apache. Perangkat lunak klien dan server bahkan dapat berada di komputer yang sama. Itu hal yang umum, seperti yang akan kita lihat.
