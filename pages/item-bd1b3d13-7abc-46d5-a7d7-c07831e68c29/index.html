Proses Umum

  1. Membuat Formulir HTML:

    • Buat form HTML dengan input-input yang sesuai untuk data siswa (nama, NIS, tanggal lahir, dll).
    • Atur atribut action pada tag <form> untuk menentukan halaman PHP yang akan memproses data formulir.
    • Atur atribut method pada tag <form> (biasanya POST untuk keamanan).

  2. Mengirim Data ke Server:

    • Ketika pengguna mengisi formulir dan mengklik tombol submit, data formulir akan dikirim ke halaman PHP yang telah ditentukan pada atribut action.

  3. Menerima Data di PHP:

    • Di halaman PHP, gunakan variabel super global $_POST untuk mengambil data yang dikirimkan dari formulir.
    • Validasi data untuk memastikan data yang diterima valid dan aman.

  4. Menyimpan Data ke Database:

    • Buat query SQL INSERT untuk memasukkan data ke dalam tabel siswa.
    • Gunakan data yang telah diambil dari formulir untuk mengisi nilai pada query SQL.
    • Eksekusi query untuk menyimpan data ke database.
