Menampilkan data dalam layar merupakan

By the end of this lesson, you should be able to...

  1. Siswa dapat menggunakan variabel, operator, dan ekspresi untuk merepresentasikan data dalam algoritma.
  2. Siswa dapat menulis kode yang efisien dan mudah dibaca.

 HTML (HyperText Markup Language): Bahasa ini digunakan untuk menstruktur konten di halaman web. HTML akan menentukan bagaimana elemen-elemen seperti judul, paragraf, gambar, dan tabel ditampilkan di browser.
PHP (Hypertext Preprocessor): Bahasa ini digunakan untuk memproses data di server. PHP bisa mengambil data dari database, mengolahnya, dan kemudian memasukkannya ke dalam struktur HTML.

DAPATKAN itu

Cara paling sederhana untuk mengirim data dari browser ke server adalah melalui URL.

Berikut URL ke halaman pada instansi Moodle OU:

  • https://moodle.oakland.edu/course/view.php?id=242965

URL-nya memiliki:

  • Protokol,https
  • Pelayannya,moodle.oakland.edu
  • Jalur halaman,/course/view.php
  • Dan beberapa hal aneh di akhir,?id=242965

Hal-hal di bagian akhir adalah string kueri . Itu ?menandakan awal string. Ada parameter dan nilai. Jadi parameter idmemiliki nilai 242965.

Anda dapat meneruskan lebih dari satu parameter, seperti ini:

  • sesuatu?nama=Rosie&kelucuan=12

Tanda ampersand (&) memisahkan argumen. Anda dapat memasukkan banyak argumen.
