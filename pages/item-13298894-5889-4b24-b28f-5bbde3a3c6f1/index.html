Basis data merupakan inti dari aplikasi web bisnis. Aplikasi menyerahkan sebagian besar pekerjaan pencarian, penambahan, dan pembaruan data ke sistem manajemen basis data (DBMS). Hal ini membuat aplikasi lebih mudah ditulis.

Basis data (DB) adalah sekumpulan data tentang sesuatu, seperti kucing-kucing di lingkungan sekitar. Data tersebut hanya tersimpan dalam satu atau beberapa berkas. Sistem manajemen basis data (DBMS) adalah perangkat lunak yang mengetahui cara membaca data dalam DB, mengubah data, menghapusnya, dan sebagainya. DBMS membuat data tersebut aktif.

Program Anda tidak akan mengakses file DB secara langsung. Sebaliknya, program Anda akan meminta DBMS untuk melakukan tugas. Itu membuat pekerjaan kita lebih mudah. ​​Mudah itu bagus.


Ada banyak DBMS, seperti SQL Server (Microsoft), Access (juga Microsoft), Oracle DBMS (dari Oracle), SQLite (terbuka)... daftarnya terus bertambah.

Server Reclaim Anda menjalankan sistem manajemen basis data MySQL. MySQL adalah DBMS sumber terbuka yang dimiliki oleh Oracle. Ada yang sejenis dengan MariaDB, yang dijalankan oleh komunitas karena memang Oracle dan kami tidak sepenuhnya mempercayainya. MySQL dan MariaDB bekerja dengan cara yang sama.

Anda juga memiliki MySQL di XAMPP. Lihat mengapa kami menggunakan XAMPP? Itu adalah hal yang sama yang ada di server ReclaimAnda. Mudah itu bagus lagi.


Satu DBMS dapat mengelola banyak basis data sekaligus. Jadi server Anda hanya memiliki satu salinan MySQL yang berjalan, tetapi menangani banyak DB.
